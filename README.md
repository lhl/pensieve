pensieve
========
Pensieve is a personal notebook/wiki.

It exists mainly because I wanted a wiki that had inline editing.


Features
--------
* Inline WYSIWYG Editing
* Automatic Nav/TOC
* Running locally/offline and syncing as well as online


Dependencies
------------
* nodejs app
* node-webkit wrapper
